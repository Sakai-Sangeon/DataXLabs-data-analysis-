# Data Analyst Internship - Task 4: SQL for Data Analysis

## Overview
This task focused on extracting and analyzing structured data from a database using SQL. The dataset used for this task is `Ecommerce_SQL_Database`. The objective was to practice writing SQL queries, joining tables, aggregating data, and creating views to derive actionable insights.

## What I Did
- Wrote SQL queries to:
  - Aggregate total revenue.
  - Calculate average revenue per customer.
  - Analyze product sales and revenue.
  - Identify recent customer purchase trends.
  - Handle missing (NULL) values.
- Created **views** to simplify complex queries and improve efficiency.
- Applied **JOINs** (INNER, LEFT) to combine data from multiple tables.
- Used **aggregate functions** (SUM, AVG, COUNT) for meaningful metrics.
- Tested and optimized queries for accuracy and performance.

## Tools
- SQLite (or MySQL/PostgreSQL)
- SQL queries (.sql file)
- Screenshots of query results

## Key Learnings
- Learned to query relational databases effectively.
- Understood how to manipulate and join multiple tables for insights.
- Practiced using aggregate functions and grouping for business metrics.
- Gained experience in handling NULL values and creating views.
- Improved skills in optimizing SQL queries for efficiency.

## Outcome
After completing this task, I am able to:
- Analyze sales and customer behavior using SQL.
- Generate actionable insights from structured data.
- Prepare datasets for further analysis or visualization.
